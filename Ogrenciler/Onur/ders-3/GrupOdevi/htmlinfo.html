1. HTML Nedir?

HTML (HyperText Markup Language), web sayfalarının yapısını tanımlamak için kullanılan bir işaretleme dilidir. Tarayıcılar, HTML kodlarını yorumlayarak içerikleri ekrana düzgün bir şekilde gösterir. Temel yapı taşları etiketlerdir ve bu etiketler açılış () ve kapanış () şeklindedir.


1.1 HTML'in Temel Yapısı

Bir HTML belgesi genelde şu bölümlerden oluşur: : Sayfanın tamamını kapsar. : Sayfa hakkında meta bilgiler (başlık, karakter seti vb.) içerir. : Görünen içeriklerin (paragraflar, resimler, videolar vb.) yer aldığı bölüm.


1.2 HTML'in Avantajları

Basitlik: HTML öğrenmesi ve kullanması kolay bir dildir. Platform Bağımsızlık: Tüm modern tarayıcılarda çalışır. Esneklik: CSS ve JavaScript ile birlikte web sayfalarını daha işlevsel ve görsel hale getirebilirsiniz. SEO Dostu: Doğru HTML kullanımı, arama motorlarında sayfanın daha iyi görünmesini sağlar.
